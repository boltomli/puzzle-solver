# 使用指南

## 快速开始

### 1. 创建项目
- 启动应用后，点击「创建新项目」
- 输入项目名称和可选描述
- 点击「创建」

### 2. 录入剧本
- 在「剧本」页面粘贴剧本文本
- 点击「保存剧本」或 Ctrl+Enter
- 首次保存时，如果 API 已配置，AI 将自动分析剧本

### 3. 分析剧本（AI）
- 点击剧本旁的「🤖 分析此剧本」按钮
- AI 将提取：人物、地点、时间段、直接事实
- 在分析结果对话框中：
  - 点击「全部添加实体」一键添加所有新人物/地点/时间段
  - 或逐个点击「添加」按钮选择性添加
  - 对于提取的事实，点击「添加到审查」加入审查队列
- 已分析的剧本显示「📊 查看分析结果」按钮，可随时回看

### 4. 管理实体
- 在「管理」页面可手动增删改：
  - 时间段（HH:MM 格式）
  - 人物（姓名、别名、描述、状态）
  - 地点（名称、别名、描述）
  - 游戏规则/提示/约束
  - 手动录入已确认的事实

### 5. 推理矩阵
- 在「矩阵」页面查看人物×时间的交叉矩阵
- 绿色 = 已确认事实，黄色 = 待审查推断
- 点击空格快速添加事实
- 「🤖 AI 推断」— 让 AI 综合所有线索推断
- 「🔄 消元推断」— 本地消元法自动推理

### 6. 审查推断
- 在「审查」页面确认或拒绝 AI/消元推断
- 按置信度排序：确定 > 高 > 中 > 低
- 接受推断自动创建事实，并触发消元连锁
- 拒绝推断需填写原因

### 7. 配置 API
- 在「设置」页面配置 OpenAI 兼容 API
- 填写 Base URL（必填）和 API Key（本地服务可留空）
- 点击「加载模型列表」获取可用模型
- 点击「测试连接」验证配置

## 工作流程

```
创建项目 → 录入剧本 → AI分析剧本 → 添加实体 → 添加事实到审查
                                                        ↓
                查看矩阵 ← AI推断/消元推断 ← 审查确认推断
                    ↓
                重复直到矩阵填满
```

## 快捷键
- Ctrl+Enter：快捷保存剧本

## 运行模式
- 桌面模式（默认）：`uv run python main.py`
- Web 模式：`uv run python main.py --web`
